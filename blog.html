What are the different ways to select an element in the DOM?
Here are the different ways to select an element in the DOM:
  • By ID – document.getElementById("id") (selects a single element by its id).
  • By Class Name – document.getElementsByClassName("class") (selects elements with a specific class).
  • By Tag Name – document.getElementsByTagName("tag") (selects elements by their tag name).
  • By CSS Selector – document.querySelector("selector") (selects the first matching element).
  • By CSS Selector (All Matches) – document.querySelectorAll("selector") (selects all matching elements).
  • By Name Attribute – document.getElementsByName("name") (selects elements by the name attribute).
What is the difference between innerHTML, innerText, and textContent ?
Here’s the main difference between innerHTML, innerText, and textContent:

1. innerHTML

  • Gets or sets HTML content inside an element.
  • Parses and renders HTML tags.
  • Example: element.innerHTML = "Hello"; (renders bold text). innerText

1. innerText

  • Gets or sets visible text inside an element.
  • Ignores hidden elements and trims whitespace.
  • Example: element.innerText = "Hello"; (only displays visible text).

1. textContent

  • Gets or sets all text, including hidden elements.
  • Does not parse HTML, treats it as plain text.
  • Example: element.textContent = "Hello"; (displays as "Hello").
What is event delegation in the DOM?

Event Delegation in the DOM is a technique where you add an event listener to a parent element instead of multiple child elements. It works because events bubble up from the target element to its ancestors.

What is event bubbling in the DOM?

Event Bubbling in the DOM is when an event starts from the target element and moves up through its parent elements.
